# phone-backend


## Technologies

* MongoDB Database(NoSQl)
* Mongoose (MongoDB framework)
* Nodejs (Express framework)



## Getting Started
### Installation
- run `npm install`
- create db
- `run npm seed` to seed data to database
- run `npm run start:dev` then navigate to `localhost:3000` on POSTMAN
